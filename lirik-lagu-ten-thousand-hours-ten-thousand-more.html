Goat is an animal. It has four legs. It eats grass, and it is always called as herbivore by human. It is also a mammal.

1. Ten Thousand Hours Divided by Six Thousand Hours is?

Jawaban :

=> 10.000 jam : 6.000 jam

=> ( 10.000 ÷ 1.000 ) jam : ( 6.000 ÷ 1.000 ) jam

=> 10 jam : 6 jam

=> ( 10 ÷ 2 ) jam : ( 6 ÷ 2 ) jam

=> 5 jam : 3 jam

=> 1 ⅔

Pembuktian !

=> 1⅔

=> 3 x 1 + 2 per 3

=> 3 + 2 per 3

=> 5/3 terbukti !
