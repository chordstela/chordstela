17. pliss lah bang kak tolong y Abang-Abang Kaka Kaka ​


Jawab:

Pada gambar tersebut, AF // KH, besar sudut ABC = 58° dan sudut BDE = 144°. Besar sudut sudut KGJ = 36⁰ dan besar sudut JGI = 86⁰. Perhatikan gambar pada lampiran: garis a // garis b, dan garis c memotong garis a dan garis b berturut-turut di titik A dan titik B.  

Sudut sehadap (besar sudutnya sama), yaitu

∠A₁ = ∠B₁

∠A₂ = ∠B₂

∠A₃ = ∠B₃

∠A₄ = ∠B₄

Sudut dalam bersebrangan (besar sudutnya sama), yaitu

∠A₂ = ∠B₃

∠A₄ = ∠B₁

Sudut luar bersebrangan (besar sudutnya sama), yaitu

∠A₃ = ∠B₂

∠A₁ = ∠B₄

Sudut dalam sepihak (jumlah sudutnya sama dengan 180⁰)

∠A₂ + ∠B₁ = 180⁰

∠A₄ + ∠B₃ = 180⁰

Sudut luar sepihak (jumlah sudutnya sama dengan 180⁰)

∠A₁ + ∠B₂ = 180⁰

∠A₃ + ∠B₄ = 180⁰

Sudut bertolak belakang (besar sudutnya sama)

∠A₁ = ∠A₄

∠A₂ = ∠A₃

∠B₁ = ∠B₄

∠B₂ = ∠B₃

Penjelasan dengan langkah-langkah:

Diketahui

AF // KH

∠ABC = 58°  

∠BDE = 144°

Ditanyakan

a. ∠KGJ

b. ∠JGI

Jawab

∠KGB = ∠ABC = 58⁰ ---> karena merupakan sudut sehadap

∠ABC + ∠ABG = 180⁰ --> karena sudut lurus

58⁰ + ∠ABG = 180⁰

∠ABG = 180⁰ – 58⁰

∠ABG  = 122⁰

∠BDE + ∠BDG = 180⁰ ---> karena sudut lurus

144⁰ + ∠BDG = 180⁰

∠BDG = 180⁰ – 144⁰

∠BDG  = 36⁰

∠BDE = ∠KGD = 144⁰ ---> karena merupakan sudut sehadap

maka

∠KGD = 144⁰

∠KGB + ∠BGD = 144⁰

58⁰ + ∠BGD = 144⁰

∠BGD = 144⁰ – 58⁰

∠BGD = 86⁰

JADI

a. ∠KGJ = ∠HGD ---> karena merupakan sudut bertolak belakang

∠HGD = ∠BDG ---> karena merupakan sudut dalam bersebrangan

berarti

∠KGJ = ∠HGD = ∠BDG = 36⁰

b. ∠JGI = ∠BGD = 86⁰ ---> karena merupakan sudut bertolak belakang
